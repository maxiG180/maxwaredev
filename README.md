# Maksym Grebeniuk - Personal Portfolio 🚀

A high-performance, single-page portfolio built for a **Full Stack .NET & React Developer**. This project showcases a modern, dark-themed aesthetic with a focus on automation and high-end animations.

## ✨ Features
- **Modern Tech Stack:** Built with **React 18**, **Vite**, and **Tailwind CSS** for maximum performance and lightning-fast deployment.
- **Cinematic Animations:** Powered by **GSAP** and **ScrollTrigger** for smooth, scroll-based "fade and slide" effects.
- **Dark Geometric Aesthetic:** A custom-styled dark theme inspired by geometric polygon textures, optimized for technical readability.
- **Fully Responsive:** Seamlessly transitions from high-resolution desktop layouts to a clean, mobile-optimized experience.
- **AI-Driven Workflow:** Developed using an advanced AI collaboration model (Gemini CLI) to streamline the architecture, design, and implementation process.

## 🛠️ Built With
- **Frontend:** React, Vite, Tailwind CSS
- **Animations:** GSAP (GreenSock Animation Platform)
- **Icons:** FontAwesome, Lucide React
- **Deployment:** Optimized for **Vercel** / **Netlify**

## 📂 Project Structure
```text
├── src/
│   ├── App.jsx          # Core UI logic & GSAP animations
│   ├── main.jsx         # React Entry point
│   └── index.css        # Tailwind directives
├── public/              # Static assets (logos, images, CV)
├── package.json         # Project dependencies
├── tailwind.config.js   # Custom theme configuration
└── vite.config.js       # Vite build configuration
```

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/maxiG180/maxwaredev.git
cd maxwaredev
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run locally
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```
